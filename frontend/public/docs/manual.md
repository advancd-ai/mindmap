## 시작하기

### 첫 로그인

1. **Google 계정으로 로그인**
   - "Sign in with Google" 버튼 클릭
   - Google 계정 선택 및 권한 승인
   - 자동으로 개인 저장소가 생성됩니다

2. **게스트 모드로 시작**
   - "Continue as Guest" 버튼 클릭
   - 별도 로그인 없이 바로 사용 가능
   - ⚠️ 게스트 모드는 모든 게스트가 공유하는 저장소를 사용합니다

### 대시보드 이해하기

대시보드는 모든 마인드맵을 관리하는 중심입니다:

- **마인드맵 목록**: 생성한 모든 마인드맵이 카드 형태로 표시됩니다
- **검색**: 상단 검색창에서 제목이나 태그로 마인드맵 검색
- **새 마인드맵 생성**: "+ Create New Mind Map" 버튼 클릭
- **마인드맵 정보**: 각 카드에 노드 수, 엣지 수, 마지막 수정 시간 표시

### 첫 마인드맵 만들기

1. 대시보드에서 **"+ Create New Mind Map"** 클릭
2. 마인드맵 제목 입력 (예: "프로젝트 계획")
3. 태그 추가 (선택사항, 쉼표로 구분)
4. **"Create Mind Map"** 클릭
5. 에디터 페이지로 자동 이동됩니다

## 기본 개념

### 마인드맵이란?

마인드맵은 아이디어를 시각적으로 구조화하는 도구입니다:

- **노드 (Node)**: 각 아이디어나 개념을 나타내는 박스
- **엣지 (Edge)**: 노드 간의 관계를 나타내는 연결선
- **루트 노드**: 마인드맵의 중심이 되는 시작 노드

### 캔버스 조작

- **줌 (Zoom)**: 마우스 휠 또는 도구 상자의 줌 버튼 사용
- **팬 (Pan)**: 배경을 드래그하여 화면 이동
- **Fit to Screen**: 모든 노드가 화면에 맞도록 자동 조정
- **Center View**: 루트 노드를 화면 중앙에 배치

## 노드 작업

### 노드 생성

#### 방법 1: 도구 상자 사용
1. 왼쪽 도구 상자에서 **"➕ Add Node"** 버튼 클릭
2. 선택된 노드가 있으면 그 근처에 생성
3. 선택된 노드가 없으면 캔버스 중앙에 생성

#### 방법 2: 키보드 단축키
- **`a`** 또는 **`A`** 키 입력
- 한글/영어 입력 상태와 무관하게 동작

#### 방법 3: 우클릭 메뉴
- 캔버스 우클릭 → **"Add Node Here"**
- 노드 우클릭 → **"Add Child Node"**

### 노드 선택

- **클릭**: 노드 클릭하여 선택
- **선택 해제**: 배경 클릭 또는 **`Esc`** 키
- 선택된 노드는 파란색 테두리로 표시됩니다

### 노드 이동

- **드래그 앤 드롭**: 노드를 클릭하고 드래그하여 이동
- 연결된 엣지는 자동으로 따라 움직입니다

### 노드 크기 조정

1. 노드 선택
2. 노드 모서리나 가장자리의 **8개 핸들** 중 하나를 드래그
3. 원하는 크기로 조정

### 노드 모양 변경

노드는 다양한 모양을 지원합니다:

- **Rectangle** (사각형): 기본 모양
- **Circle** (원형): 강조가 필요한 노드
- **Diamond** (다이아몬드): 의사결정 포인트
- **Hexagon** (육각형): 프로세스 단계
- **Cloud** (구름): 아이디어나 개념
- **Capsule** (캡슐): 액션 아이템
- **File** (파일): 문서나 파일 참조
- **Card** (카드): 정보 카드

**변경 방법**:
1. 노드 우클릭
2. **"Change Shape"** 선택
3. 원하는 모양 선택

### 노드 접기/펼치기

- 노드 우측 상단의 **"‹"** 버튼 클릭
- 접힌 노드는 자식 노드와 연결선이 숨겨집니다
- 다시 클릭하면 펼쳐집니다

### 노드 삭제

1. 삭제할 노드 선택
2. **`d`** 또는 **`D`** 키 입력 (또는 **`Delete`** / **`Backspace`**)
3. 확인 다이얼로그에서 "확인" 클릭
4. ⚠️ 연결된 엣지도 함께 삭제됩니다

### 노드 편집

노드 편집은 세 가지 방법이 있습니다:

#### 1. 더블 클릭 (인라인 편집)
- 노드를 더블 클릭하면 노드 위에서 직접 편집
- 팝업 없이 빠르게 수정 가능
- **`Alt + Enter`** 저장, **`Esc`** 취소

#### 2. 편집 버튼 (모달 편집)
- 노드 선택 후 나타나는 **"✏️ 편집"** 버튼 클릭
- 큰 편집 창이 열립니다
- 긴 텍스트나 서식이 많은 내용 편집에 적합

#### 3. 키보드 단축키
- 노드 선택 후 **`e`** 또는 **`E`** 키 입력
- 모달 편집 모드로 진입

## 엣지(연결선) 작업

### 엣지 생성

#### 방법 1: 노드 핸들 사용
1. 노드 선택
2. 노드 가장자리의 **파란색 핸들** 클릭
3. 연결할 다른 노드로 드래그
4. 대상 노드에서 마우스 버튼 놓기

#### 방법 2: 도구 상자 사용
1. 시작 노드 선택
2. 왼쪽 도구 상자에서 **"🔗 Connect"** 버튼 클릭
3. 연결할 대상 노드 클릭
4. 자동 연결 모드로 연결됩니다

#### 방법 3: 우클릭 메뉴
1. 노드 우클릭
2. **"Connect"** 선택
3. 대상 노드 클릭

### 엣지 선택

- **클릭**: 엣지 선을 클릭하여 선택
- 선택된 엣지는 두꺼운 선으로 표시됩니다

### 엣지 타입 변경

엣지는 세 가지 타입을 지원합니다:

- **Straight** (직선): 간단한 연결
- **Curved** (곡선): 부드러운 연결
- **Bezier** (베지어 곡선): 복잡한 곡선

**변경 방법**:
1. 엣지 우클릭
2. **"Edge Type"** 선택
3. 원하는 타입 선택

### 엣지 레이블 추가

1. 엣지 더블 클릭 또는 우클릭 → **"Add Label"**
2. 레이블 텍스트 입력
3. 레이블은 엣지 중앙에 표시됩니다

### 엣지 삭제

1. 삭제할 엣지 선택
2. **`d`** 또는 **`D`** 키 입력 (또는 **`Delete`** / **`Backspace`**)
3. 확인 다이얼로그에서 "확인" 클릭

## 편집 모드

Open Mindmap은 세 가지 편집 모드를 제공합니다:

### 1. Text 모드 (인라인 편집)

**특징**:
- 빠른 텍스트 수정에 적합
- 노드 위에서 직접 편집
- 서식 지원 없음

**사용 방법**:
- `contentType: 'text'`인 노드 더블 클릭
- 또는 노드 헤더의 "📄 Text Editor" 버튼 클릭

**단축키**:
- **`Enter`**: 저장 후 종료 (단일 줄)
- **`Alt + Enter`**: 저장 후 종료 (다중 줄)
- **`Esc`**: 취소

### 2. Rich Editor 모드 (WYSIWYG 편집)

**특징**:
- 서식이 있는 텍스트 편집 (Bold, Italic, Underline, Link 등)
- 이미지, 링크 등 리치 콘텐츠 지원
- 큰 편집 창 제공 (모달 또는 인라인)

**사용 방법**:
- 노드 더블 클릭 → 인라인 편집
- 노드 선택 후 편집 버튼 클릭 → 모달 편집
- 도구 상자의 **"📝 Rich Editor"** 버튼 클릭

**단축키**:
- **`Alt + Enter`**: 저장 후 종료
- **`Ctrl/Cmd + B`**: Bold
- **`Ctrl/Cmd + I`**: Italic
- **`Ctrl/Cmd + U`**: Underline
- **`Ctrl/Cmd + K`**: Link 삽입/편집
- **`Shift + Ctrl/Cmd + V`**: 플레인 텍스트 붙여넣기

### 3. Markdown Editor 모드

**특징**:
- 마크다운 문법으로 구조화된 텍스트 작성
- 코드 블록, 테이블 등 고급 문법 지원
- 실시간 미리보기 (Split View)

**사용 방법**:
- 노드 더블 클릭 → 인라인 편집
- 도구 상자의 **"📄 Markdown Editor"** 버튼 클릭 → 모달 편집

**단축키**:
- **`Alt + Enter`**: 저장 후 종료
- **`Tab`**: 2칸 들여쓰기

**마크다운 문법 예시**:
```markdown
# 제목 1
## 제목 2
**굵게** *기울임*
- 리스트 항목 1
- 리스트 항목 2
```

### 모드 전환

1. **도구 상자 사용**: 노드 선택 후 도구 상자의 편집 모드 버튼 클릭
2. **노드 헤더 버튼**: Text 모드 노드의 헤더에 표시되는 버튼 클릭
3. **자동 선택**: 노드의 `contentType`에 따라 적절한 모드가 자동 선택됩니다

## 도구 상자 (Toolbox)

도구 상자는 에디터 왼쪽에 위치한 플로팅 패널입니다.

### 위치 및 디자인

- **위치**: 에디터 왼쪽 중앙
- **디자인**: Apple 스타일의 반투명 배경과 블러 효과
- **고정**: 스크롤과 무관하게 항상 같은 위치에 고정

### 도구 구성

#### 1. 노드 생성 도구
- **➕ Add Node**: 새 노드 추가
- 선택된 노드가 있으면 그 근처에 생성
- 선택된 노드가 없으면 캔버스 중앙에 생성

#### 2. 선택 도구
- **👆 Select**: 선택 모드 활성화

#### 3. 연결 도구
- **🔗 Connect**: 노드 연결 모드 활성화
- 노드 선택 시에만 표시
- 자동 연결 모드로 동작

#### 4. 편집 도구
- **✏️ Edit**: 선택된 노드 편집 모드 진입
- 노드 선택 시에만 표시

#### 5. 삭제 도구
- **🗑️ Delete**: 선택된 노드/엣지 삭제
- 노드/엣지 선택 시에만 표시

#### 6. 모양 변경 도구
- **🔷 Shape**: 노드 모양 변경
- 노드 선택 시에만 표시

#### 7. 임베드 도구
- **📎 Embed**: YouTube 비디오나 웹 페이지 임베드
- 노드 선택 시에만 표시

#### 8. 줌 및 뷰 제어
- **🔍 Zoom In**: 확대
- **🔍 Zoom Out**: 축소
- **📐 Fit to Screen**: 모든 노드가 화면에 맞도록 조정
- **🎯 Center View**: 루트 노드를 화면 중앙에 배치

### 동적 표시

도구 상자는 현재 선택 상태에 따라 동적으로 표시됩니다:

- **노드 미선택**: 기본 도구만 표시 (Add Node, Select, Zoom 등)
- **노드 선택**: 편집, 삭제, 연결, 모양 변경 등 추가 도구 표시
- **엣지 선택**: 엣지 관련 도구만 표시

## 키보드 단축키

### 노드 조작

| 단축키 | 동작 |
|--------|------|
| `a` / `A` | 노드 추가 ✅ |
| `e` / `E` | 선택된 노드 편집 모드 진입 ✅ |
| `d` / `D` | 선택된 노드/엣지 삭제 ✅ |
| `Delete` / `Backspace` | 선택된 노드/엣지 삭제 ✅ |
| `Enter` | 선택된 노드 인라인 편집 모드 진입 ✅ |

### 엣지 조작

| 단축키 | 동작 |
|--------|------|
| `d` / `D` | 선택된 엣지 삭제 ✅ |
| `Delete` / `Backspace` | 선택된 엣지 삭제 ✅ |

### 편집 단축키

| 단축키 | 동작 |
|--------|------|
| `Alt + Enter` | 저장 후 종료 |
| `Ctrl/Cmd + Enter` | 저장 |
| `Esc` | 취소 |

### 리치 에디터 포맷팅

| 단축키 | 동작 |
|--------|------|
| `Ctrl/Cmd + B` | Bold |
| `Ctrl/Cmd + I` | Italic |
| `Ctrl/Cmd + U` | Underline |
| `Ctrl/Cmd + K` | Link |
| `Shift + Ctrl/Cmd + V` | 플레인 텍스트 붙여넣기 |

### 캔버스 조작

| 단축키 | 동작 |
|--------|------|
| `Esc` | 선택 해제 / 상태 초기화 |
| `Ctrl/Cmd + V` | 이미지 붙여넣기 (캔버스 포커스 시) |

### 빠른 워크플로우

#### 빠른 노드 추가 및 편집
1. **`a`** 키 입력 → 새 노드 추가
2. 자동으로 선택된 새 노드에서 **`e`** 키 입력 → 편집 모드 진입
3. 내용 수정
4. **`Alt + Enter`** → 저장 후 종료

#### 빠른 편집
1. 노드 클릭 → 노드 선택
2. **`Enter`** 키 입력 → 인라인 편집 모드 진입
3. 내용 수정
4. **`Alt + Enter`** → 저장 후 종료

> **참고**: 키보드 단축키는 한글/영어 입력 상태와 무관하게 동작합니다 (`e.code` 사용). IME 입력 중(한글/중국어/일본어 입력 중)일 때는 단축키가 무시됩니다.

자세한 내용은 [키보드 단축키 가이드](./keyboard-shortcut.md)를 참조하세요.

## 고급 기능

### 이미지 작업

#### 이미지 노드 생성

**방법 1: 이미지 붙여넣기**
1. 클립보드에 이미지 복사 (예: 스크린샷, 이미지 파일)
2. 캔버스 포커스 상태에서 **`Ctrl/Cmd + V`** 입력
3. 자동으로 이미지 노드가 생성됩니다

**방법 2: 이미지 파일 업로드**
1. 노드 우클릭 → **"Add Image"**
2. 이미지 파일 선택
3. 업로드 후 이미지 노드로 변환

#### 이미지 표시

- 이미지 노드는 자동으로 이미지 크기에 맞게 조정됩니다
- 이미지 노드도 일반 노드처럼 이동, 크기 조정, 연결 가능합니다

### 임베드 콘텐츠

#### YouTube 비디오 임베드

1. 노드 선택
2. 우클릭 → **"Add Embed"** → **"YouTube"**
3. YouTube URL 입력 (예: `https://www.youtube.com/watch?v=...`)
4. 비디오가 노드 내에 임베드됩니다

#### 웹 페이지 임베드

1. 노드 선택
2. 우클릭 → **"Add Embed"** → **"Web Page"**
3. 웹 페이지 URL 입력
4. 웹 페이지 미리보기가 노드 내에 표시됩니다

### 텍스트 정렬

노드 내 텍스트의 정렬을 조정할 수 있습니다:

**수평 정렬**:
- **Left** (왼쪽)
- **Center** (가운데)
- **Right** (오른쪽)

**수직 정렬**:
- **Top** (상단)
- **Middle** (중앙)
- **Bottom** (하단)

**변경 방법**:
1. 노드 우클릭
2. **"Text Align"** 선택
3. 수평/수직 정렬 선택

### PDF 내보내기 및 보기

#### PDF 보기

1. 에디터 헤더의 **"👁️ PDF 보기"** 버튼 클릭
2. 현재 마인드맵이 PDF로 렌더링됩니다
3. 새 탭에서 PDF 미리보기

#### PDF 특징

- 모든 노드와 연결선이 포함됩니다
- 이미지와 임베드 콘텐츠도 포함됩니다
- 한글 폰트 지원 (Noto CJK)
- 공유 페이지에서도 PDF 보기 가능

## 공유 및 협업

### 마인드맵 공유

#### 공유 링크 생성

1. 에디터 헤더의 **"🔗 Share"** 버튼 클릭
2. 공유 설정 선택:
   - **Read-only** (읽기 전용): 다른 사용자는 보기만 가능
   - **Editable** (편집 가능): 다른 사용자도 편집 가능
3. 공유 링크 복사
4. 링크를 다른 사용자에게 전달

#### 공유 페이지 특징

- 공유 링크를 통해 접근 가능
- 로그인 없이도 볼 수 있습니다 (읽기 전용 모드)
- PDF 보기 기능 지원
- 도구 상자에서 줌, Fit to Screen, Center View 사용 가능

### 게스트 모드

게스트 모드는 로그인 없이 바로 사용할 수 있는 모드입니다:

- **장점**: 빠른 시작, 별도 계정 불필요
- **단점**: 모든 게스트가 공유하는 저장소 사용
- **권장**: 테스트나 임시 작업에 적합

게스트 모드에서 생성한 마인드맵은:
- 다른 게스트 사용자도 볼 수 있습니다
- Google 로그인 후 개인 저장소로 마이그레이션 가능 (향후 지원 예정)

## 저장 및 동기화

### 자동 저장

- 마인드맵은 **GitHub**에 자동으로 저장됩니다
- 각 마인드맵은 별도의 Git 브랜치에 저장됩니다
- 변경사항은 Git 커밋으로 기록됩니다

### 수동 저장

1. 에디터 헤더의 **"💾 Save (Preview)"** 버튼 클릭
2. JSON Preview 다이얼로그에서 변경사항 확인
3. **"Confirm & Save"** 클릭
4. GitHub에 저장됩니다

### 저장소 구조

```
Repository: {username}/mindmap-data

Branches:
├── main                     # 문서 (README.md)
├── maps/map_1234567890     # 마인드맵 1
│   └── map.json
├── maps/map_9876543210     # 마인드맵 2
│   └── map.json
└── maps/map_5678901234     # 마인드맵 3
    └── map.json
```

### 버전 관리

- 각 저장마다 버전 번호가 자동으로 증가합니다
- Git 히스토리를 통해 이전 버전 확인 가능
- 푸터에 현재 버전 번호 표시

### 동기화

- 여러 기기에서 동일한 Google 계정으로 로그인하면 동일한 마인드맵에 접근 가능
- 실시간 동기화는 현재 지원하지 않습니다 (향후 지원 예정)
- 저장 후 다른 기기에서 새로고침하면 최신 버전이 표시됩니다

## 문제 해결

### 일반적인 문제

#### Q: 노드가 보이지 않아요
**A**: 
- **Fit to Screen** 버튼을 클릭하여 모든 노드를 화면에 맞춥니다
- 줌 레벨을 조정해보세요 (도구 상자의 줌 버튼 사용)

#### Q: 이미지가 표시되지 않아요
**A**:
- 이미지 파일 크기가 10MB 이하인지 확인하세요
- 지원 형식: PNG, JPEG, GIF, WebP
- 공유 페이지에서는 공유 토큰이 필요할 수 있습니다

#### Q: 저장이 안 돼요
**A**:
- 인터넷 연결을 확인하세요
- GitHub 저장소 설정이 올바른지 확인하세요
- 브라우저 콘솔에서 오류 메시지를 확인하세요

#### Q: 키보드 단축키가 동작하지 않아요
**A**:
- 텍스트 입력 중일 때는 단축키가 동작하지 않습니다
- IME 입력 중(한글/중국어/일본어 입력 중)일 때는 단축키가 무시됩니다
- 노드나 엣지가 선택되어 있는지 확인하세요

#### Q: PDF에 한글이 표시되지 않아요
**A**:
- 서버에 한글 폰트가 설치되어 있는지 확인하세요
- Docker 환경에서는 `font-noto-cjk` 패키지가 설치되어 있어야 합니다

### 성능 문제

#### 큰 마인드맵이 느려요
**A**:
- 노드를 접어서 표시되는 노드 수를 줄이세요
- 불필요한 노드나 엣지를 삭제하세요
- 브라우저 캐시를 지우고 새로고침하세요

### 접근성

#### 스크린 리더 사용
- 모든 버튼과 노드에 ARIA 레이블이 포함되어 있습니다
- 키보드만으로 모든 기능을 사용할 수 있습니다

#### 고대비 모드
- 다크 모드 지원 (향후 지원 예정)
- 고대비 테마 자동 적용 (향후 지원 예정)
